## 🛠️ Install project

1. Clone the repository

```bash
git clone https://github.com/Megas-MDN/product-search.git
```

2. Enter the cloned folder

```bash
cd product-search
```

3. Install the dependencies

```bash
npm install
```

4. Build the project

```bash
npm start
```

5. Run in development mode

```bash
npm run dev
```
